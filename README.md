## ✨ Features

- 🌤️ **Real-time Weather** - Current conditions for any city worldwide
- 📍 **Geolocation Support** - Auto-detects your location for instant weather
- 🔍 **Smart Search** - Autocomplete with city suggestions
- 📊 **3-Day Forecast** - Weather predictions with rain probability
- 💡 **Smart Advisory** - Actionable advice (umbrella, sunscreen, etc.)
- 🎨 **Dynamic Backgrounds** - Weather-based scenic backgrounds
- 📱 **Fully Responsive** - Works on all devices (mobile, tablet, desktop)
- 🚀 **Blazing Fast** - Optimized with OpenWeather API

## 🛠️ Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Glass-morphism, animations, responsive design
- **JavaScript (ES6+)** - Async/await, DOM manipulation, API integration
- **OpenWeather API** - Weather data and geocoding
- **Font Awesome** - Icons
- **Unsplash** - Dynamic background images
